## Adding a Document

1. Drop the file into `.agents/docs/`.
2. Add a row to the index table above with the filename and a one-line description.
3. Reference it in a feature plan under **Background / Context** so agents know to read it.

## Document Types

- **API specs** — Third-party or internal API documentation
- **Design docs** — Architecture decision records, design mockups
- **Research notes** — Findings from spikes or explorations
- **External references** — Copied or summarized external documentation relevant to the project
